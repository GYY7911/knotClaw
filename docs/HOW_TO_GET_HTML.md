# 如何手动获取DeepSeek对话HTML

## ⚠️ 重要说明

DeepSeek是一个**单页应用（SPA）**，对话数据是通过JavaScript动态加载的。

**直接保存的HTML（Ctrl+S 或 Ctrl+U）不包含对话数据！**

这是因为：
1. Ctrl+U 获取的是原始HTML，不包含JavaScript渲染后的内容
2. Ctrl+S 保存的也是原始HTML骨架
3. 对话数据是在页面加载后通过API动态获取并渲染的

## ✅ 正确的获取方法

### 方法一：使用自动获取脚本（推荐）

运行项目中的辅助脚本：

```bash
python fetch_html.py
```

然后输入DeepSeek分享链接，脚本会使用Selenium获取渲染后的完整HTML。

### 方法二：使用浏览器开发者工具获取渲染后的HTML

1. **打开对话链接**
   - 在浏览器（Chrome/Edge/Firefox）中打开DeepSeek分享链接
   - 例如：`https://chat.deepseek.com/share/xxxxx`

2. **等待页面完全加载**
   - 确保所有对话内容都已显示
   - 页面不再有加载动画

3. **查看源代码**
   - 按 `Ctrl + U`（Windows/Linux）
   - 或 `Cmd + Option + U`（Mac）
   - 或右键点击页面 → "查看网页源代码"

4. **复制全部内容**
   - 按 `Ctrl + A` 全选
   - 按 `Ctrl + C` 复制

5. **保存或粘贴**
   - 保存到文件（如 `deepseek.html`）
   - 或直接粘贴到程序中

## 方法二：使用开发者工具

1. **打开对话链接并等待加载完成**

2. **打开开发者工具**
   - 按 `F12`
   - 或 `Ctrl + Shift + I`
   - 或右键 → "检查"

3. **获取HTML**
   - 在 "Elements" 标签页
   - 右键点击最顶部的 `<html>` 标签
   - 选择 "Copy" → "Copy outerHTML"

4. **保存或粘贴到程序中**

## 方法三：保存完整网页

1. **打开对话链接并等待加载完成**

2. **保存网页**
   - 按 `Ctrl + S`
   - 选择 "网页，完整" 或 "网页，仅HTML"
   - 保存到本地

3. **在程序中选择从文件读取**
   - 输入保存的HTML文件路径

## 验证HTML是否正确

正确的HTML应该包含以下内容之一：

- `__NEXT_DATA__` - Next.js数据
- `"messages":` - 消息数组
- `"conversation":` - 对话数据

如果HTML中包含这些内容，程序就能正确解析对话。

## 常见问题

### Q: 为什么无法自动获取？
A: DeepSeek使用AWS WAF（Web Application Firewall）保护网站，需要JavaScript验证。普通的HTTP请求无法通过验证。

### Q: 粘贴后提示"未找到对话数据"？
A: 请确保：
- 页面已完全加载
- 复制了完整的源代码
- 不是复制了渲染后的页面内容

### Q: 可以使用其他浏览器吗？
A: 可以，Chrome、Edge、Firefox、Safari都可以。
